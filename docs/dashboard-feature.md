# Dashboard 数据概览功能

## 功能概述

Dashboard（数据概览）是一个全新的视图，通过可视化图表展示任务管理系统中各个维度的统计信息，帮助用户快速了解任务完成情况和工作进展。

## 主要功能

### 1. 关键指标卡片

在页面顶部展示5个核心指标：

- **总任务数**：系统中所有任务的总数
- **已完成**：已完成的任务数量
- **进行中**：正在进行中的任务数量
- **延期任务**：已过期但未完成的任务数量
- **完成率**：已完成任务占总任务的百分比

每个指标都配有相应的图标和颜色标识，便于快速识别。

### 2. 可视化图表

#### 任务状态分布（饼图）
- 展示待安排、进行中、已完成三种状态的任务分布
- 使用不同颜色区分各状态
- 显示具体数量和百分比

#### 优先级分布（柱状图）
- 展示四象限中未完成任务的分布情况
- Q1（重要且紧急）- 红色
- Q2（重要不紧急）- 蓝色
- Q3（紧急不重要）- 橙色
- Q4（不紧急不重要）- 灰色

#### 最近7天完成趋势（折线图）
- 展示过去7天每天完成的任务数量
- 帮助用户了解工作效率趋势
- 绿色曲线表示完成情况

#### 本周任务分布（柱状图）
- 展示本周（周一到周日）每天未完成任务的数量
- 帮助用户合理安排本周工作

### 3. 详细统计表格

以卡片形式展示四个象限的详细统计：
- 显示每个象限的任务数量
- 包含象限说明文字
- 使用对应的颜色主题

## 技术实现

### 依赖库

- **recharts**：用于绘制各类图表（饼图、柱状图、折线图）
- **date-fns**：用于日期处理和格式化
- **lucide-react**：提供图标组件

### 组件结构

```
components/
  └── DashboardView.tsx  # Dashboard主组件
```

### 数据计算

Dashboard组件使用 `useMemo` 钩子优化性能，只在任务数据变化时重新计算统计信息：

- 实时统计各类指标
- 动态计算图表数据
- 自动过滤和分组任务

## 使用方法

1. 在侧边栏点击"数据概览"按钮
2. 查看各项统计指标和图表
3. 通过图表了解任务分布和完成趋势
4. 根据数据调整工作计划

## 响应式设计

Dashboard采用响应式布局：

- **移动端**：单列布局，图表垂直排列
- **平板**：2列网格布局
- **桌面端**：多列网格布局，充分利用屏幕空间

## 颜色方案

统一的颜色系统确保视觉一致性：

- 红色系：紧急/延期任务
- 蓝色系：进行中/第二象限
- 绿色系：已完成任务
- 橙色系：第三象限
- 灰色系：第四象限/待安排
